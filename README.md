# Netflix Clone Project

A responsive front-end recreation of the Netflix landing page.

## Description

This project is a clone of the Netflix homepage, built using HTML and CSS. It features a responsive design that works on both desktop and mobile devices, and includes an interactive FAQ accordion section.

## Features

- **Responsive Design:** Adapts to different screen sizes.
- **Interactive FAQ:** Accordion style FAQ section using pure CSS (checkbox hack).
- **Modern Styling:** Uses CSS gradients, flexbox, and absolute positioning.

## Technologies Used

- HTML5
- CSS3

## How to View

Simply open the `index.html` file in any modern web browser.
